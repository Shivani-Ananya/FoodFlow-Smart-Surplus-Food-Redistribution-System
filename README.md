# FoodFlow-Smart-Surplus-Food-Redistribution-System
FoodFlow is an interactive web-based platform designed to reduce food waste by connecting surplus food providers (restaurants, canteens) with hunger zones (NGOs, shelters) in real time.
This project goes beyond a simple map — it simulates a smart food logistics system with role-based access, real-time interaction, and intelligent matching.

🎯 Problem Statement

Large amounts of edible food are wasted daily, while many communities struggle to access basic meals. The lack of coordination between food providers and receivers leads to inefficiency and missed opportunities for redistribution.

💡 Solution
FoodFlow provides a real-time map-based system that:
Identifies surplus food locations
Highlights hunger zones
Enables quick coordination between stakeholders
Suggests optimal redistribution routes

🔑 Key Features
🔐 Role-Based Login
🟢 Food Provider (can add surplus food)
🔴 Food Receiver (can view and request food)
🚴 Volunteer (can accept and complete deliveries)

🗺️ Interactive Chennai Map
Green markers → Food available
Red markers → Food needed
Clickable markers with detailed information

⚡ Smart Matching System
Automatically finds the best match between food sources and hunger zones
Suggests nearest routes and optimal distribution

📍 Route Visualization
Displays delivery paths between locations
Simulates real-time logistics planning

🔄 Live Status Tracking
Each location updates dynamically:
🟢 Available
🟡 Reserved
⚫ Completed

🔔 Notification Simulation
“NGO notified”
“Pickup reserved”
“Delivery completed”

⏳ Expiry & Urgency System
Shows time remaining before food expires
Urgent needs highlighted with priority markers

📊 Impact Dashboard
Tracks system impact in real time:

🍱 Meals saved
🌍 Food waste reduced
📍 Active locations

🌡️ Heatmap Layer
Visualizes food surplus density vs hunger zones
Helps identify high-need areas

🚨 SOS Mode
Allows emergency food requests
Displays high-priority alerts on the map

🛠️ Tech Stack
HTML, CSS, JavaScript
Leaflet.js (Maps)
OpenStreetMap
